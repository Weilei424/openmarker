# Code of Conduct

OpenMarker is committed to providing a welcoming and respectful environment for all contributors.

This project follows a simple principle:

Be respectful and constructive.

---

# Our Standards

Examples of behavior that contributes to a positive community:

• Being respectful of differing viewpoints  
• Providing constructive feedback  
• Helping other contributors learn  
• Focusing on improving the project  

Examples of unacceptable behavior include:

• Harassment or discrimination  
• Personal attacks  
• Trolling or insulting comments  
• Publishing private information without consent

---

# Scope

This Code of Conduct applies to:

• GitHub issues  
• Pull requests  
• Project discussions  
• Community interactions related to the project

---

# Enforcement

Project maintainers are responsible for enforcing these rules.

Maintainers may take actions including:

• Removing inappropriate content  
• Temporarily blocking contributors  
• Banning users from the project

These actions are taken to maintain a healthy community environment.

---

# Reporting Issues

If you experience or witness unacceptable behavior, please report it by opening a private issue or contacting the project maintainers.

Reports will be handled respectfully and confidentially.
